# ShiftMatch - シフト管理システム

配送業界向けのシフト管理・日報システム

## 主な機能

- シフト管理
- 日報提出・承認
- メンバー管理
- チャット機能
- 通知機能
- ルート管理
- 単価設定
- ダッシュボード

## 技術スタック

- Next.js 15.5.4
- TypeScript 5
- Prisma 6.16.3
- PostgreSQL (Neon)
- NextAuth.js 4
- Tailwind CSS 3
- Radix UI

## 開発環境のセットアップ

```bash
# 依存パッケージのインストール
npm install

# データベースのセットアップ
npm run db:push

# 開発サーバーの起動
npm run dev
```
